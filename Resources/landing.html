What is this website?

This website is going to be depicting visualizations of weather data for a multitude of random cities around the world. Such visualizations will include graphs like Latitude vs. Cloudiness, vs. Max Temperature, vs. Humidity, and vs. Wind Speed. A separate page will be created for each of these graphs. In addition, a page will be created to view all of them at the same time, and a page will be created to show a table of the data used to create these graphs.
